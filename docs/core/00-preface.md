# 序章　なぜ文明は創発しなくなったのか

21世紀の文明は、かつてないほど精密に設計され、
かつてないほど創発しなくなった。
都市は整流され、経済は自動化され、個人は快適に孤立している。

創発とは、秩序と混沌のあわいで生じる「生の跳躍」である。
それは新しい関係、新しい物語、新しい意味を生み出す現象だ。
だが現代社会では、すべてが「安全に最適化」され、
その跳躍が抑制されている。

原因は明白だ。
人間社会は、部分最適と還元主義によって構成されている。
経済は利益という単一指標に還元され、
政治は制度効率に、
倫理は規範遵守に、
教育は試験点数に。

これらはそれぞれ合理的である。
しかし、それらが連結した全体は非合理的だ。
なぜなら、それらが接続する回路の「温度」を誰も測っていないからだ。

本書は、この欠落を埋める試みである。
あらゆるシステムは熱をもち、秩序と無秩序の境界で創発する。
その創発を失えば、文明は静かに熱死する。

われわれは、再び「生まれる世界」を取り戻さねばならない。
そのために必要なのは、新しい知識体系ではなく、
**「存在の熱力学」**としての一般システム論の再定義である。

本書が問うのはこうだ。

> 世界は、なぜ創発をやめたのか。
> そして、どうすれば再び揺らぎ始めるのか。
