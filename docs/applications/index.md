# 応用 - 理論を現実に重ねてみる

創発文明論で考えた枠組みを、身近なシステムに当てはめてみるとどう見えるか。ここではソフトウェア開発、AI、教育、企業という4つの領域を取り上げている。

どれも「こうすべきだ」という処方箋ではなく、システム論の視点で眺めたときに浮かび上がる構造の話だ。理論が現実とどこで噛み合い、どこでずれるのか -- その手触りを残しておきたいと思って書いた。
