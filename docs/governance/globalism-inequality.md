# グローバリズムと格差 - 国内循環の構造的変化

## 核心的主張
**グローバリズムは国内のマネーフローを破壊し、格差を構造的に生み出す**

## 国内経済循環の基本

### 健全な経済循環
```
A => B => C => A
```
- 国内でマネーが循環
- GDPが増大
- A、B、C全員が豊かになる
- これが本来の経済循環

### グローバリズムによる変化
```
A => B => C => a（海外の安いサービス）
```
- 国内循環が細る
- マネーが海外流出
- Aの収入減少 → Bの収入減少 → Cの収入減少
- デフレスパイラルの一因

### 重要な洞察
> 多少高いカネを払ってもそのカネは回りまわって帰ってくるはずだった

**これが国内経済循環の本質だ**

## 格差拡大のメカニズム

### 労働者の価格競争
```
日本の労働者 vs 賃金1/10の海外労働者
性能差：せいぜい2-3倍
価格差：10倍
結果：日本の労働者が不利に
```

### 企業の戦略
- 日本：消費地として定義
- 海外：生産地として定義
- 利益：企業に蓄積
- 賃金：海外に流出

### この構造の問題
企業と海外はWinを得たが、国内の労働者はLoseの側に置かれた。「Win-Win」という表現の裏には、こうした構造的な非対称がある。

## システム価値論での分析

### 日本システムの価値毀損
```
システム価値 = 創発 × 存続期間
```
- 国内循環の破壊 → 創発力の低下
- 格差拡大 → 社会不安 → 存続期間の短期化
- グローバリズムは日本のシステム価値を下げている

## 価値の序列の変化

### 現在の序列
```
先端技術 ＞ 投資 ＞ 労働
```
- 労働の価値：最低
- 投資の価値：中間
- 技術の価値：最高

### 問題点
- 労働者が大多数
- しかし価値は最も低い
- 民主主義との間に矛盾が生まれている

## 国家安全保障の視点

### コロナ禍で露呈した構造
- マスク生産を中国に依存
- 危機時に供給が滞る
- 国民の生命が危機に
- これが「効率化」の帰結のひとつ

### 安全保障の原則
```
国内生産 = 安全保障
海外依存 = 脆弱性
```

### 地産地消の意味
> 地産地消は「安全保障上当たり前の戦略」

- 地域内循環 → 地域の豊かさ
- 国内循環 → 国家の豊かさ

## 通貨政策の構造的問題

### 通貨安戦略の側面
```
通貨安 = 実質的な労働者の賃金抑制
```

### なぜ企業は通貨安を求めるか
- 輸出競争力の向上 → 表の理由
- 労働コストの実質的な抑制 → 構造的な結果
- 日本の労働力を相対的に安く使える

### 根本的な問い
> 通貨を安くしないと売れないようなサービスは、価格競争力の低いサービス

そうした分野に注力し続ける必要があるのかという問いが生まれる。

## 先端人材の流出

### 構造的な流れ
- 優秀な人材 → アメリカへ
- 理由：給与が高い
- 日本：通貨安で給与が相対的に低い
- 結果：人材流出

### 悪循環
```
通貨安 → 低賃金 → 人材流出 → 競争力低下 → さらなる通貨安
```

## 年金制度の構造的課題

### 年金の構造
年金は本質的に、後の世代が前の世代を支える仕組みになっている。
- 前提：人口ピラミッドが安定していること
- 現実：逆ピラミッド化が進行
- 結果：制度の持続可能性に疑問が生じている

もちろん、積立方式への移行や支給年齢の引き上げなど、制度を延命させる手段はある。しかし、構造的な前提が崩れていることは見つめる必要がある。

### 皮肉な結果
- 年金で老後の安心を確保 → 子供の経済的必要性が低下
- 少子化が進む → 年金の支え手が減る
- 制度が自らの前提を掘り崩す構造になっている

## 民主主義への影響

### 人口構造の現実
> 60歳超の世代が人口の大きな割合を占めている

### 何が起きるか
```
生産年齢人口を超えた世代が国家の方向に大きな影響力を持つ
```
これは民主主義にとって構造的な危機である。

### 結果責任の非対称
- 高齢世代：あと10-20年
- 若い世代：あと50-60年
- でも投票力は同じ
- 結果責任と権利の間にずれがある

### 平均余命投票制との接続
```
投票力 = 結果責任期間 = 平均余命
```
この結果責任の非対称を是正する仕組みとして、[平均余命投票制とデジタル民主主義](./democracy-redesign.md)を提案している。投票力を結果責任に比例させることで、世代間の公平を構造的に担保する。

## 自由 vs 平等のジレンマ

### 根本的対立
- 自由 → 不平等
- 平等 → 不自由

### 日本の変遷
- 過去：平等重視（護送船団）
- 現在：自由重視（競争）
- 結果：格差拡大

### グローバリズムの構造
```
グローバリズム = 自由な競争 = 不平等の拡大
```

## 考えられる方向性

### 1. 関税の見直し
- 国内産業保護
- マネー流出の抑制
- 雇用確保

### 2. 国内循環の優先
- 多少高くても国内から買う
- 長期的には戻ってくる
- 全体最適

### 3. 通貨高戦略
- 優秀な人材を集める
- 購買力向上
- 真の豊かさ

### 4. 民主主義の改革
- 結果責任に基づく投票（[平均余命投票制](./democracy-redesign.md)）
- デジタル民主主義
- 世代間公平

## システム論的な整理

### グローバリズムの価値
```
価値 = 創発 × 存続期間
創発：格差により低下
存続：社会不安により短期化
結論：価値は低い
```

### 国内循環の価値
```
価値 = 創発 × 存続期間
創発：全体の豊かさから創発
存続：安定により長期化
結論：価値は高い
```

## 最終的な洞察

グローバリズムは、
効率の名の下に、
国内経済の循環構造を変えた。

企業は利益を得て、
労働者は貧しくなり、
国家は衰退する。

必要なのは、
国内循環の再構築。
それが長期的な豊かさへの道だ。
