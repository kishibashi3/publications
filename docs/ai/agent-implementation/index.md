## シリーズの特徴

- **思想レベルから実装まで一貫**：抽象論で終わらず、実務で使える構造を示す
- **フレームワーク非依存**：Dify、LangGraph、AgentEngine、どれでも通用する
- **破綻パターンから逆算**：「なぜ壊れるか」から「壊れない構造」を導く
- **エージェントOS**：文脈・実行・時間を分離する統合構造論
